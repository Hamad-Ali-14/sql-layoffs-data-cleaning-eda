# Layoffs Data Cleaning and EDA Project

## Project Overview

This project focuses on cleaning and analyzing the World Layoffs dataset using MySQL.

## Dataset

* Original Dataset: layoffs.csv
* Cleaned Dataset: layoffs_cleaned.csv

## Data Cleaning Tasks

* Removed duplicate records
* Standardized text values
* Converted date column to DATE datatype
* Handled null values
* Removed unnecessary columns

## Exploratory Data Analysis

* Total layoffs by company
* Total layoffs by industry
* Total layoffs by country
* Total layoffs by year
* Monthly layoffs trends
* Rolling total layoffs
* Top companies by year
